# Trend Confirmation System

Trend confirmation keeps SwiftChart from reacting to every bounce or pullback.

## Confirmation inputs

SwiftChart looks at EMA alignment, slope, RSI pressure, ADX, and recent structure.

## Bull trend confirmation

Bullish confirmation improves when price holds above key averages, reclaims resistance, and prints higher structure.

## Bear trend confirmation

Bearish confirmation improves when price breaks support, forms lower structure, and momentum agrees with downside.
