# Performance Tracking

SwiftChart saves signal history so the system can be reviewed after the fact.

## Signal memory

Saved ideas include direction, entry, stop, targets, score, regime, confidence, and result state.

## Outcome checks

The history system can review later candles to see whether entry, take profit, or stop levels were reached.

## Why it matters

Performance tracking turns the scanner into a measurable workflow instead of a stream of one-off opinions.
