# Signal Engine

The signal engine scores potential long and short setups using the same core categories: regime alignment, zone quality, liquidity behavior, momentum, volume, and risk/reward.

## Scoring model

Each candidate begins with market context. A with-trend setup receives more weight than a counter-trend setup. Range trades are allowed only near strong edges, while mid-range ideas are rejected.

## Acceptance rules

A setup must clear the minimum score after regime adjustment. It also needs a valid risk/reward profile and a clear invalidation level.

## Rejection rules

Rejected signals are logged with the current bias, missing condition, and reason. This makes it easier to audit why the scanner waited instead of forcing a trade.
