# Alert System

The alert system sends SwiftChart signals to Telegram when a setup clears the strategy threshold.

## Alert quality

Alerts are meant to be selective. The scanner waits for clean setups rather than posting every small candle reaction.

## Telegram flow

Users can subscribe to alerts and receive concise signal summaries with direction, entry, stop, targets, confidence, and invalidation.

## Future controls

Planned controls include watchlists, subscription tiers, per-timeframe alerts, and quiet hours.
